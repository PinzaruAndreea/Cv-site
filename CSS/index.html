For CSS (Cascading Style Sheet) we had to create a project from scratch. The HTML code was mainly provided by the person who created the course. I had to create the CSS code and a little bit of JavaScript code. Firstly, I learned about the behaviour of the elements in a page depending on the viewport. The course started from basic CSS properties to more complex ones, such as Flexbox and CSS Grid.

We first began with creating the home page. The navigation bar was one of the first elements I made, this will be displayed on all pages. Alternatively, for mobile users, I created a toggle bar that opens a div to display the page navigation options. On the homepage there is also a plan option which prompts you to a form page created through JavaScript to sign up for the desired plan.

Next, we have the customers and packages pages, the CSS code provides a multitude of functionalities. I added a few animations on both + one on the nav bar.

This project taught me many useful stuff, one being a good comprehension on how to make the elements behave a certain way and how to design them to your heart contempt.
